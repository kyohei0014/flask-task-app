# タスク管理アプリ

## 概要
Python と Flask、SQLite を使ったシンプルなタスク管理アプリです。  
タスクの追加・編集・削除・完了切替が可能で、期限も設定できます。  
チェックボックスやタスク行クリックで完了状態を切り替え、完了タスクは薄く表示されるUIを実装しています。

## 技術スタック（アプリやシステムを作るために使っている技術の組み合わせ）
- Python 3 / Flask
- SQLite
- HTML / CSS / Jinja2

## 学んだこと・工夫した点
- Webアプリケーションの基本構造（ルーティング、テンプレート、DB操作）の理解
- CRUD（作成・読み取り・更新・削除）の実装
- UI/UXの改善（チェックボックスの位置調整、完了タスクの視覚化、編集フォームで期限変更）
- GitHub によるソースコード管理
- 初心者でも読みやすいコード設計を意識
- タスク、期限の入力欄を位置を見やすいようにコードの書き直しなどの修正

## 使い方
1. Python と Flaskをインストール
2. `python app.py` を実行
3. ブラウザで `http://127.0.0.1:5000` にアクセス
4. タスクの追加、編集、完了状態切替を操作可能
